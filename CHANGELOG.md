## [1.0.0] - 2026-02-03

### 新增 (Added)
- 🎉 项目初始化，由 **AME & Entertainment** 开发并开源发布
- 💳 支付宝聚合支付（PC网站 / H5 / 扫码）
- 💚 微信支付（Native 扫码 / H5）
- 📦 统一订单管理（查询 / 详情 / 关闭）
- 🏪 多商户管理（创建 / 编辑 / API密钥管理）
- 💰 提现审核流程（申请 → 审核 → 打款）

- 🔔 支付回调自动通知商户（含重试机制）
- 🐳 Docker Compose 一键部署（Nginx + MySQL + API）
- ▲ Vercel Serverless 部署支持
- ☁️ Cloudflare Workers + Pages 部署支持
- ⚡ GitHub Actions CI/CD 自动化流水线
- 📝 完整 API 文档 + 部署文档 + FAQ
