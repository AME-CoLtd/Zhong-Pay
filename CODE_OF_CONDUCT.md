# 行为准则 / Code of Conduct

## 我们的承诺

为了营造一个开放和包容的环境，我们（贡献者和维护者）承诺：无论年龄、体型、残疾、民族、性别认同、经验水平、国籍、外貌、种族、宗教或性取向如何，参与我们项目的每个人都不会受到骚扰。

## 我们的标准

**有助于创造积极环境的行为包括：**

- 使用友好包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对社区其他成员表示同理心

**不可接受的行为包括：**

- 使用性语言或图像，以及不受欢迎的性关注或挑逗
- 恶意评论、人身攻击或政治攻击（Trolling）
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如地址）
- 在专业场合被认为不适当的其他行为

## 执行责任

项目维护者（AME & Entertainment）有责任澄清可接受行为的标准，并对任何不可接受的行为采取适当和公平的纠正措施。

## 联系方式

如果发生骚扰等违规行为，请通过 [提交 Issue](https://github.com/AME-dev/zhong-pay/issues/new?labels=question&title=%5BConduct%5D+) 联系项目维护团队（AME & Entertainment），我们会私下跟进处理。
