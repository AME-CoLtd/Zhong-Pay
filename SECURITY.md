# 安全策略 / Security Policy

## 支持的版本 / Supported Versions

| 版本 / Version | 支持状态 / Support Status |
|----------------|---------------------------|
| latest (main)  | ✅ 积极维护 / Actively maintained |
| < 1.0.0        | ❌ 不再支持 / No longer supported |

## 报告安全漏洞 / Reporting a Vulnerability

**⚠️ 请勿通过公开 Issue 报告安全漏洞！**

如果你发现了安全漏洞，请通过以下方式**私下报告**：

1. **GitHub Private Vulnerability Reporting（首选）**：[提交安全报告](https://github.com/AME-dev/zhong-pay/security/advisories/new)
2. **GitHub Issues（安全类）**：[提交 Security Issue](https://github.com/AME-dev/zhong-pay/issues/new?labels=security&template=bug_report.yml&title=%5BSecurity%5D+)

### 报告内容请包含

- 漏洞描述及潜在影响
- 详细复现步骤
- 受影响的版本
- 建议的修复方案（如有）

### 响应时间

| 阶段 | 预期时间 |
|------|----------|
| 确认收到报告 | 48 小时内 |
| 初步评估结果 | 7 个工作日内 |
| 修复并发布补丁 | 视严重程度，30 天内 |

### 致谢

我们会在安全公告中对负责任披露的研究者表示感谢。
